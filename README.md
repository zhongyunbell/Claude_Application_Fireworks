## Claude application examples

# Website:

https://zhongyunbell.github.io/Claude_Application_Fireworks/


# Prompt:

Build an interactive fireworks show as a single HTML file. When I click anywhere on the screen, launch a firework from the bottom to that point and explode it into colorful particles with realistic gravity and fade. Use Van Gogh's Starry Night painting as the background (load it from Wikimedia Commons). Include 5 explosion types: spherical burst, ring, double ring, willow, and crackle (with secondary sub-explosions). Add a button bar on the right side to select the explosion type or set it to random. Include an "Auto Show" button that continuously launches fireworks at random positions, and a "Grand Finale" button that rapid-fires 25 fireworks. Use 10 different color palettes. Add a credit line at the bottom: "Created by Julie Huang zhongyunbell@gmail.com". Make sure UI button clicks don't trigger firework launches behind them.
